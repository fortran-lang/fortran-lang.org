## __Name__

__char__(3) - \[CHARACTER\] Character conversion function
(GFDL)

## __Syntax__
```fortran
result = char(i, kind)
   elemental integer function char(i,kind)

    integer(kind=KIND),intent(in) :: c
    integer,intent(in),optional :: KIND
```
## __Description__

__char(i, kind)__ returns the character represented by the integer __i__.

## __Arguments__

  - __i__
    : The type shall be _integer_.

  - __kind__
    : (Optional) An _integer_ initialization expression indicating the kind
    parameter of the result.

## __Returns__

The return value is of type _character_

## __Examples__

Sample program:

```fortran
program demo_char
implicit none
integer :: i = 74
character(1) :: c
    c = char(i)
    print *, i, c ! returns 'J'
end program demo_char
```
  Results:
```text
             74 J
```
